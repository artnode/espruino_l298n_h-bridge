# espruino_l298n_h-bridge
Espruino: Driving two DC motors or one stepper motor using the l298n h-brige

---------------------------------------------------------------

USAGE (DC-motor):

motorDC(1); //full speed

motorDC(-1); //reverse full speed

motorDC(0.5); //half speed

motorDC(0); //stop

---

USAGE (stepper-motor):

L298Nstep(100);

L298step(-350); //350 steps reverse
